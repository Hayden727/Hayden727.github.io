About

I work on machine learning systems — making large language models serve faster and learn better. At RedNote I work on LLM Infra, focused on inference acceleration and reinforcement learning for large-scale model serving.

I'm a core committer of sglang-omni, a multi-stage pipeline framework for omni models, and a contributor to SGLang, where I designed and implemented FDFO — a First-Done-First-Out scheduling mode for diffusion-LLM inference that lets a batch stop waiting for its slowest request. Earlier I worked on MLIR/LLVM compiler infrastructure and performance engineering, which still shapes how I think about systems today.
